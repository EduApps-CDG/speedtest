Okay, but what is a Megabit?

In computing, the processor uses a binary system (1 and 0), we give each of these numbers the name of bit. the number of bits is given by the number of digits. When we have an 8-bit number (8 digits), we call that number byte. When we have 1000 bytes, we call them Kilobytes, and finally, when we have 1000 Kilobytes, we change them to Megabytes and so on.
